Experimental GLB assets from the project Google Drive GLB folder.

These files are reviewed runtime test fixtures for the lazy Three.js viewer:

- `dino.glb`: Draco-compressed GLB.
- `humanoid-2.glb`: meshopt-compressed GLB with KTX2 textures.
- `car-paint.glb`: Draco-compressed GLB.

Keep this folder for measured viewer/asset-pipeline validation. Do not replace production portfolio models with additional Drive exports without size, license, and visual QA checks.
