# Card thumbnails — /assets/cards/

Файлы для превью на карточках в сайдбаре.

## Структура

```
assets/cards/
  orbital-mk-ii.svg   # 800 × 600 рекомендуется
  vega-shell.svg
  ...
  nyx-panther.svg     # Organic
  drift-koi.svg
  glint-owl.svg
```

## Замена

1. Положить файл с **тем же именем** (kebab-case, латиница) и расширением `.svg`, `.webp` или `.jpg`.
2. Путь жёстко прописан в `js/main.js` → `CARDS_DATA[id].thumb`. Поменяй туда — и превью обновится.
3. Если файла нет — карточка покажет градиент + лейбл-плейсхолдер.

## Рекомендации

- Размер: **800 × 600** (4:3), до 200 KB
- Формат: `.webp` (лучшее сжатие) / `.svg` (векторные абстракции) / `.jpg` (fallback)
- Имя в формате kebab-case, совпадает с `data-id` карточки
