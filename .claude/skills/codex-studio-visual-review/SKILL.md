# Codex Studio Visual Review

Do not judge visual quality from source alone. Require either:

- a screenshot path, or
- a local URL that can be opened and captured.

If neither is provided, stop and ask for one of those inputs. Do not infer visual quality from source code.

Assess whether the view reads as a senior 3D professional portfolio rather than a generic template. Focus on type, density, contrast, sidebar utility, specific project signals, and absence of generic decorative patterns.

For visual changes, verify code first and then review the rendered page.
