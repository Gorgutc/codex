## ✅ Берём — Референсы для вдохновения

### Уровень и ощущение

Эти сайты задают планку качества и настроения:

| Сайт | Что берём |
|---|---|
| https://atlab.io | Тёмная эстетика, детализированные 3D-объекты, студийный уровень |
| https://bruno-simon.com | Интерактивность, playful 3D — дух экспериментов |
| https://lusion.co | Тёмная эстетика, WebGL визуалы, полное погружение |
| https://activetheory.net | Кинематографичность, детальная анимация |
| https://grilli.typeface.com | Типографическая строгость, воздух, минимализм |
| https://stripe.com | Чистота кода, продуманность деталей, доверие |

### Детали, которые берём

- **Scroll-driven анимации:** появление элементов при скролле через GSAP ScrollTrigger
- **Типографические заголовки:** крупный display-шрифт на тёмном фоне
- **Full-bleed секции:** изображения на всю ширину экрана
- **Плавные переходы:** fade + translate, без резких скачков
- **Визуальный ритм:** разная высота секций, нет монотонности
- **Кастомный курсор** (опционально, если не замедляет): небольшое dot с hover-реакцией

## ❌ Не берём — Запрещённые паттерны

### «AI-эстетика» — то, что выдаёт сгенерированный дизайн

| Паттерн | Почему нет |
|---|---|
| Фиолетово-синие градиенты | Выглядит как SaaS-шаблон 2021 года |
| Светящиеся орбы и неоновые блобы | Декоративный мусор без смысла |
| 3 одинаковые карточки с иконкой в кружке | Самый узнаваемый AI-layout |
| Gradient на кнопках | Дешевле, чем solid accent |
| Emoji в заголовках | Моментально снижает восприятие уровня |
| «Ваш надёжный партнёр в дизайне» | Generic-копирайт без смысла |
| Centered всё подряд | Центрирование заголовков, описаний, карточек — шаблон |
| Одинаковые секции по высоте | Монотонный ритм без визуального дыхания |
| Цветные border-left на карточках | Узнаваемый AI-паттерн |
| Иконки в цветных кружках | Немедленно узнаётся как AI-генерация |

### Конкретные запреты для этого проекта

- ❌ Волнистые SVG-разделители между секциями
- ❌ Плавающие геометрические фигуры как «декорации»
- ❌ Светлый режим как основной (dark mode — главный)
- ❌ Стоковые фото людей (Unsplash-портреты)
- ❌ Счётчики «500+ клиентов», «10 лет опыта» без доказательств
- ❌ Кириллица в UI-тексте (сайт полностью на английском)
- ❌ Анимированные gradient-фоны (Aurora, Mesh gradient)

## 🔍 Проверочный вопрос

Перед добавлением любого элемента:
> «Это добавляет смысл или просто заполняет пространство?»

Если ответ «заполняет пространство» — убрать.
