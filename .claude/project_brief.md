# Project Brief — Codex Studio
> Главный документ проекта — цель, аудитория, позиционирование, стиль

---

## 🎯 Цель проекта

Создать портфолио-сайт 3D-дизайнера уровня Senior.
Сайт строится с помощью AI (Perplexity Labs/Computer) без ручного написания кода.
Код должен быть чистым, production-ready и готовым к деплою через GitHub Pages или Netlify из VSCode.

**Главная задача сайта:**
Показать уровень и стиль автора зарубежным клиентам и работодателям.
Сайт = живое портфолио + точка контакта.

**Критерий успеха:** посетитель за 5 секунд понимает — это серьёзный профессионал.

---

## 👤 Аудитория

**Первичная:**
- Арт-директора и креативные директора в студиях и агентствах (международный рынок)
- Продуктовые компании и стартапы — 3D-визуализация для прототипирования и маркетинга
- Игровые студии и продакшн-компании, нанимающие 3D-художников

**Вторичная:**
- Коллеги-дизайнеры (профессиональное сообщество)
- Потенциальные партнёры по фрилансу

**Что думает посетитель при заходе:**
> «Это студия или один человек? Можно ли доверить серьёзный проект?»

**Что он должен почувствовать:**
> «Это профессионал. Хочу посмотреть работы. Хочу написать.»

## 🧭 Позиционирование

| Параметр | Значение |
|---|---|
| Специализация | Hard Surface 3D |
| Уровень | Senior — не Junior или Mid |
| Подход | Качество, излишняя детализированность |
| Локация | Удалённо, для всего мира |
| Ключевое слово | Precision (точность) |
| Язык сайта | English — только (международный рынок) |

**Голос бренда:** уверенный, немногословный, профессиональный.
Никаких «я люблю создавать» и «страсть к дизайну» — только конкретика и качество.

---

## 🎨 Визуальный стиль

**Общее ощущение:** тёмный, дорогой, точный.
Уровень atlab.io, lusion.co — визуал на первом месте, минимум лишних слов.

**Цветовая температура:** холодная нейтральная база + один точечный акцент (стальной синий `#327AAE` или тил `#2E8B8F`).

**Типографика:** контраст между крупным display-шрифтом (Clash Display) и чистым sans-serif (General Sans).

**Плотность:** умеренная. Много воздуха, но не пусто. Секции намеренно разной высоты.

**Режим:** dark mode — основной, light mode — переключаемая опция.

## 🏗 Технический стек

```
HTML + CSS + Vanilla JS          ← без фреймворков
GSAP v3 + ScrollTrigger          ← через CDN
Clash Display + General Sans     ← через Fontshare CDN
GitHub Pages / Netlify           ← деплой статики
```

**Структура файлов** — строго по `structure.md`.

## 🔒 Неизменяемые решения

- Шрифты — только Clash Display + General Sans
- Язык контента — только English
- Файловая структура — только по `structure.md`
