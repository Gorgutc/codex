# Agent Docs

This directory is the Codex-native operating manual for Codex Studio.

- `architecture.md`: factual stack and runtime map
- `orchestration.md`: when and how to use agents
- `code_review.md`: code audit and cleanup rules
- `verification.md`: how to verify changes
- `preview-contract.md`: production-facing card and asset preview rules
- `industrial-editorial-refresh-next-session.md`: portable handoff for the next refresh session
- `skill-map.md`: project-local Codex skills
- `audit-summary.md`: outcome of the initial full-project audit
- `instruction-audit.md`: current instruction and workflow drift findings
- `archive_policy.md`: how to retire stale Claude-era material
- `migration-from-claude.md`: compatibility notes
- `plan_template.md`: long-task planning template
- `evals/`: prompts that test future agent behavior
- `adrs/`: decisions that should not drift
