# Long Task Plan Template

Use this for large audits, refactors, instruction rewrites, or multi-agent work.

## Goal

State the concrete outcome and how success will be verified.

## Baseline

- Current branch and git status.
- Current `npm run verify` result.
- Files or subsystems in scope.

## Orchestration

List agents to run in parallel and the specific question each must answer.

## Implementation

Describe file groups, edit order, and what is intentionally out of scope.

## Verification

List commands, manual checks, screenshots, or eval prompts.

## Final Artifacts

List reports, docs, code changes, archived material, and residual risks.
